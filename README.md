# AI Movie & Series Recommender

This is a strict modular monorepo for the AI-powered movie and series recommender system.

## Project Structure

- `apps/web`: Next.js frontend
- `apps/api`: Fastify backend
- `packages/*`: Shared modules with strict domain boundaries

## Requirements

- Node.js 20+
- pnpm 9+
- PostgreSQL
- Redis

## Setup

```bash
# Install dependencies
pnpm install

# Start development servers
pnpm dev
```
